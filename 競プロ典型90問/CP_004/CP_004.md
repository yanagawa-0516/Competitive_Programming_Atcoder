# CP 004 - Cross Sum

[競プロ典型90問 004](https://atcoder.jp/contests/typical90/tasks/typical90_d)  

・2025/9/20 回答

# 解説

グリッドに対して $i$ 行目の総和、 $j$ 列目の総和をそれぞれ計算しておく。あとは各マスについての答えとして該当する行と列のものを足せばよい。
ただ交差部分(求める部分のマスの値)は重複しているので一回引くのを忘れないように！\
\
計算量は "各行、各列の総和を求める時間" $+$ "各マスの出力を計算する時間" で $\mathrm{O}(HW)$ になる。
